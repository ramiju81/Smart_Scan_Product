Control de Vencimientos

Sistema inteligente de lectura por código de barras y QR.

Cargar productos (CSV)

1 Escanear / ingresar código

Usa la cámara, una pistola lectora o digita el código manualmente.

Compatible con códigos de barras, QR y lectores tipo pistola.

¿Qué hace este sistema?

  • Lee códigos de barras y QR desde cámara o lector.
  • Consulta una base de datos local con productos y fechas.
  • Muestra una ficha emergente con toda la información del producto.
  • Envía notificaciones periódicas de productos vencidos o por vencer.

Alertas automáticas

Cada 10 minutos se revisan los productos cargados y, si hay vencidos o próximos a vencerse (3, 2 o 1 día), el sistema envía notificaciones emergentes por cada producto.

Mantén esta página abierta (puede estar minimizada). No es necesario hacer clic en ningún botón para que las alertas funcionen.

Sistema desarrollado como prototipo de control de vencimientos.
